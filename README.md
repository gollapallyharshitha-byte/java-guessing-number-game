# Guessing Number Game (Java)

A simple Java GUI-based number guessing game built using Java Swing.

## Features
- Random number generation
- User-friendly graphical interface
- Attempt counter
- High/Low hints

## Technologies Used
- Java
- Java Swing

## How to Run

Compile:
javac GuessingGame.java

Run:
java GuessingGame
